# 代码注释

## 疑问：#的注释

* **问**：加`#`号后机器能读取这个命令吗？
* **答**：不能

Python中一行的开头是`#`，后面的内容就会被（解释器/编译器）忽略了

->专业术语叫做 **注释**=**注释代码**：仅仅是起解释和说明，供人类看的内容 机器不读，会忽略掉

->不同编程语言的注释代码的语法都不同，比如：

* 普遍采用的（C/C++/Java等）：
    * 单行注释：`// xxx`
    * 多行注释：`/* yyy */`
      * 其中`yyy`可以是很多行
* `Python`： `# xxx`
* `HTML`：`<!-- xxx -->`
* `ini`文件：`; xxx`
